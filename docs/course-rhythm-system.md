# 《设计概论》课程节奏系统｜course-rhythm-system

## 1. 页面节奏规范（全课统一）
- 单页结构上限：1个核心结论 + 1个证据图 + 1个互动触发点。
- 文本上限：正文不超过6行，超过即拆页。
- 视觉焦点：每页仅1个主焦点，避免信息并列竞争。

## 2. 慢节奏页 / 快节奏页规则
### 慢节奏页（概念、定义、总结）
- 时间：2.5–4分钟/页
- 结构：结论句 → 概念框 → 教师解释
- 使用场景：章节开头、方法讲解、阶段小结

### 快节奏页（案例、讨论、冲突）
- 时间：1–2分钟/页
- 结构：问题抛出 → 快速判断 → 即时反馈
- 使用场景：课堂提问、对照页、错误诊断页

## 3. 认知冲击页规则
- 每10页至少1张认知冲击页。
- 必须包含：反直觉命题 + 对应证据。
- 标题格式："你以为…，其实…" / "看起来…，但…"。
- 目标：打断惯性认知，迫使学生重建判断标准。

## 4. 案例页规则
- 案例页四件套：问题现场 / 设计策略 / 结果数据 / 可迁移点。
- 禁止只放美图不讲结果。
- 每个案例至少给1个失败风险点，避免单向成功叙事。

## 5. AI未来页规则
- 禁止空泛口号（如“AI颠覆一切”）。
- 必须回答三件事：
  1) AI参与了哪个环节；
  2) 提升了哪个指标；
  3) 哪个责任仍由设计师承担。
- 推荐图表：能力-责任矩阵、工作流泳道图、前后对比柱图。

## 6. 总结页规则
- 每一段总结只回答3句：
  1) 今天确定了什么标准；
  2) 这些标准如何用于项目；
  3) 下一节将解决什么新问题。
- 结尾必须有“宣言式一句话”。

## 7. 40分钟课堂时间分配逻辑
- 开场（10%）：4分钟
- 教材知识线（50–60%）：20–24分钟
- 文明线（20–25%）：8–10分钟
- AI线（10–15%）：4–6分钟
- 总结与作业（5–10%）：2–4分钟

## 8. 课堂互动频率规则
- 最低频率：每8–10分钟至少一次互动。
- 互动类型轮换：投票 → 找错 → 小组判断 → 口头复述 → 快速写卡。
- 互动回答规范：必须包含“对象+场景+判断依据”，不接受空泛观点。

## 风格声明
本系统用于把课程从“内容堆叠”升级为“导演式教学”：
- 以设计文明为叙事骨架，
- 以教材知识为方法核心，
- 以AI时代能力为前瞻延展。
