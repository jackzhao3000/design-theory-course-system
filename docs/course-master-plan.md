# 《设计概论》课程总控系统｜course-master-plan

> 世界观定位：未来学术感 + 设计文明叙事 + AI时代课程导演系统  
> 课程对象：职业本科·环境艺术设计  
> 课程总目标：把学生从“做图者”训练为“空间系统决策者”

## 一、8次课整体结构总览
1. 第1课：设计是什么——从造型到问题解决
2. 第2课：功能、形式与系统——设计判断的三把尺
3. 第3课：人、场景与行为——用户研究与空间证据
4. 第4课：材料、结构与建造——从概念到可落地
5. 第5课：城市更新与公共性——存量时代的设计责任
6. 第6课：生态与可持续——全生命周期设计方法
7. 第7课：AI协同设计工作流——生成、分析、模拟、验证
8. 第8课：综合项目路演——方法论、价值观与职业定位

## 二、逐课导演脚本（8次课）

### 第1课：设计是什么
- 核心问题：设计是审美表达，还是问题解决？
- 教材知识线：设计定义、设计要素、设计流程总览。
- 设计文明线：从手工艺到工业设计的角色迁移。
- AI未来线：AI生成图出现后，设计师角色为何更重要。
- 关键案例：校园导视系统“好看但难用”的失败样本。
- 课堂互动：30秒找错；1句话定义设计。
- 认知冲击点：**“效果图正确，不代表设计正确。”**
- 宣言式结尾：**“设计不是装饰，它是组织现实的方法。”**

### 第2课：功能、形式与系统
- 核心问题：如何建立可执行的设计判断标准？
- 教材知识线：功能指标、形式认知、系统协同。
- 设计文明线：标准化、现代主义、多元化的逻辑演进。
- AI未来线：AI能力边界与设计责任边界。
- 关键案例：徐汇滨江、校园微更新、医院导流仿真。
- 课堂互动：A/B图比较、指标补全、任务卡分类。
- 认知冲击点：**“会生成方案，不等于会做判断。”**
- 宣言式结尾：**“设计价值=功能×形式×系统。”**

### 第3课：人、场景与行为
- 核心问题：没有用户证据，方案为何不可信？
- 教材知识线：观察法、访谈法、行为路径与痛点定义。
- 设计文明线：大众社会中的用户观念演变。
- AI未来线：用AI做访谈整理、行为聚类、情景模拟。
- 关键案例：地铁换乘空间导流优化。
- 课堂互动：现场行为地图速绘；痛点优先级排序。
- 认知冲击点：**“没有证据的共情，只是想象。”**
- 宣言式结尾：**“先理解人，再定义空间。”**

### 第4课：材料、结构与建造
- 核心问题：为什么很多好概念无法落地？
- 教材知识线：材料性能、节点逻辑、施工与维护约束。
- 设计文明线：建造技术如何重塑空间美学。
- AI未来线：AI辅助构造推演、成本估算、节点校核。
- 关键案例：公共空间座椅系统从概念到施工图。
- 课堂互动：材料替换挑战；节点容错判断。
- 认知冲击点：**“不可建造的美，不是专业方案。”**
- 宣言式结尾：**“设计要对建造负责。”**

### 第5课：城市更新与公共性
- 核心问题：更新是为了商业，还是为了公共生活质量？
- 教材知识线：更新类型、利益相关者、公共性评估。
- 设计文明线：增量城市到存量城市的治理转向。
- AI未来线：AI支持街区活力评估与时段热力预测。
- 关键案例：成都玉林街区、社区口袋公园改造。
- 课堂互动：公共性评分；冲突协商角色扮演。
- 认知冲击点：**“可进入不等于可使用，可使用不等于公平。”**
- 宣言式结尾：**“城市更新首先更新的是公共关系。”**

### 第6课：生态与可持续
- 核心问题：可持续是附加项，还是设计底线？
- 教材知识线：生命周期、碳足迹、运营维护与回收。
- 设计文明线：生态文明视角下的环境设计伦理。
- AI未来线：AI进行能耗模拟、材料替代、碳排比选。
- 关键案例：海绵校园中庭改造方案。
- 课堂互动：碳排路径拆解；减碳策略竞赛。
- 认知冲击点：**“不考虑运营的绿色，只是一次性绿色。”**
- 宣言式结尾：**“可持续不是风格，是底线。”**

### 第7课：AI协同设计工作流
- 核心问题：AI进入流程后，哪些环节必须由人主导？
- 教材知识线：问题定义、方案生成、验证迭代的职责划分。
- 设计文明线：工具演进史中的专业权力再分配。
- AI未来线：多工具协同、提示词工程、评估闭环。
- 关键案例：校园共享中庭15分钟概念冲刺。
- 课堂互动：人机分工表；提示词调参对比。
- 认知冲击点：**“快不是价值，判断才是价值。”**
- 宣言式结尾：**“设计师不是提示词操作者，而是系统导演。”**

### 第8课：综合项目路演
- 核心问题：如何证明你的方案对现实真正有效？
- 教材知识线：论证结构、图板叙事、评估指标。
- 设计文明线：专业表达如何进入公共讨论。
- AI未来线：AI辅助演示、问答预演、风险检查。
- 关键案例：学生组项目路演与复盘。
- 课堂互动：同伴评审；一问一答压力测试。
- 认知冲击点：**“能讲清楚，才算真正理解。”**
- 宣言式结尾：**“你的作品，必须经得起现实追问。”**

## 三、8次课递进关系（避免重复）
- 1→2：从“设计定义”进入“设计判断标准”。
- 2→3：从“标准”进入“证据来源（人和行为）”。
- 3→4：从“需求洞察”进入“建造可行性”。
- 4→5：从“单体空间”进入“城市公共系统”。
- 5→6：从“公共性”进入“长期生态责任”。
- 6→7：从“方法底线”进入“AI协同提效与边界”。
- 7→8：从“流程能力”进入“综合论证与职业表达”。

## 四、课程世界观（统一叙事）
1. 设计不是对象生产，而是关系组织。
2. 文明演进决定设计议题，技术进步改变设计工具。
3. AI放大效率，但不会替代价值判断与责任承担。
4. 职业设计教育的核心是：问题定义能力 + 系统决策能力 + 公共伦理能力。
