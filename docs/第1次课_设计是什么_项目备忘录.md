# 项目备忘录｜第1次课：设计是什么？设计从哪里来？

## 一、项目基本信息

**课程主题**：第1次课｜设计是什么？设计从哪里来？——从人类造物到现代设计专业  
**对应教材**：绪论 + 第1章“设计的历程”  
**课程定位**：建立整门课的设计本质认知、历史认知与学习路径。  
**建议成果形态**：50页课程型PPT，强调大标题、强判断、少段落、图像叙事与课堂互动。  

## 二、本讲核心目标

1. 建立学生对“设计不是单纯美化，而是解决问题、组织体验与表达价值”的基本判断。
2. 通过人类造物史理解设计从生存工具、手工艺、工业生产到信息与AI时代的演进。
3. 建立观察设计的五个角度：功能、形式、体验、文化、社会价值。
4. 为后续“设计门类与设计体系”课程建立知识入口和学习路径。

## 三、整体叙事线

本讲从“设计离我们有多近”进入，以日常物、工具与界面建立学生的感性连接；随后回到人类造物的起点，说明设计从生存问题中产生；再展开设计的多重功能，帮助学生突破“设计只是好看”的误区；中段以设计发展阶段为主线，串联自发设计、自觉设计、手工艺时代、工业时代、现代设计、信息时代与AI时代；最后通过椅子、手机等案例回到当代，完成“设计始终围绕人”的核心判断。

## 四、视觉与版式原则

- **总体气质**：米白、深灰为主，搭配克制的强调色；避免过度装饰。
- **文字策略**：大标题、短判断、少段落；每页只解决一个核心问题。
- **图像策略**：使用“石器、陶器、现代椅、数字界面”的时间纵深拼贴作为贯穿性视觉母题；案例页使用具体器物或界面图像。
- **课堂节奏**：概念页负责判断，功能分析页负责解释，图文案例页负责证据，互动页负责转化为学生任务。

## 六、通用讲稿规范

每页讲稿建议遵循以下五步，不必逐字重复，但要保持课堂语言一致：

1. 用一句判断引入页面标题。
2. 说明该页在本讲结构中的位置。
3. 结合画面案例解释核心概念。
4. 提醒学生从功能、形式、体验、文化、社会价值五个角度观察。
5. 用一句过渡语连接下一页。

## 八、后续制作建议

1. **先统一母版**：优先完成A-H八类模板，再逐页填充内容，避免50页风格漂移。
2. **先做关键页**：建议先制作第1、4、19、31、38、43、48页，用这些页确定视觉气质和叙事节奏。
3. **图片需分层管理**：通用拼贴图用于概念页和功能分析页，案例图用于石器、陶器、礼器、明式家具、包豪斯、用户体验、AI等重点页。
4. **讲稿避免复读页面文字**：讲稿应补充判断、举例和过渡，而不是把页面标题再解释一遍。
5. **互动页要留时间**：第46-47页建议安排5-8分钟，让学生将前面概念转化为实际判断。
