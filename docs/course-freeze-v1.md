## 1. 视觉系统冻结

### 1.1 字体
- 中文标题：思源黑体 Heavy（备选：HarmonyOS Sans SC Bold）
- 中文正文：思源黑体 Regular
- 英文与数字：Inter / SF Pro Display
- 字号层级：
  - H1：72–96
  - H2：48–60
  - H3：32–40
  - Body：24–28
  - Caption：18–20

### 1.2 标题层级
- H1：章节命题（结论先行）
- H2：问题展开（Why/How）
- H3：执行动作（动词开头）
- 标题优先级：结论 > 问题 > 方法 > 数据

### 1.3 页面节奏
- 单页结构：1结论 + 1证据 + 1互动触发
- 正文不超过6行，超出即拆页
- 每2页至少1次视觉切换（全图/对照/图表）

### 1.4 深色背景体系
- 主背景：#0B1020
- 次背景：#111A2E
- 分割线：#27324A
- 主文字：#F5F7FF
- 次文字：#A8B3CF
- 强调色：#6EE7FF / #7C3AED / #22C55E / #F59E0B

### 1.5 图像风格
- 关键词：future academic cinematic / keynote stage / dark minimal
- 构图：单焦点 + 大留白 + 中远景优先
- 禁止：高饱和霓虹堆叠、无信息价值科幻UI、卡通化人物

### 1.6 信息图结构
- 标准图谱：
  1) 三环图（功能/形式/系统）
  2) 双轴图（技术推进×价值重估）
  3) 泳道图（学生/AI/用户反馈）
  4) 矩阵图（能力×责任）
  5) 对照表（问题/策略/结果/迁移）

## 2. 课程结构冻结（三线并行）

### 2.1 教材知识线
- 职能：给出可执行方法与可测标准
- 输出：定义、指标、流程、验证

### 2.2 设计文明线
- 职能：给出历史逻辑与价值坐标
- 输出：时代背景、公共性、伦理转向

### 2.3 AI未来线
- 职能：给出工具边界与职责边界
- 输出：人机分工、效率指标、责任归属

### 2.4 三线关系（冻结）
- 教材线是方法底盘
- 文明线是价值框架
- AI线是能力延展
- 任何一课都必须三线同题推进，不允许单线漂移

## 3. 页面结构冻结

### 3.1 慢节奏页（概念/定义/总结）
- 时长：2.5–4分钟/页
- 模板：结论句 → 概念框 → 教师解释

### 3.2 快节奏页（冲突/提问/判断）
- 时长：1–2分钟/页
- 模板：问题抛出 → 快速判断 → 即时反馈

### 3.3 案例页
- 必含四件套：问题现场 / 设计策略 / 结果数据 / 可迁移点
- 禁止只给美图，不给结果

### 3.4 AI未来页
- 必答三问：
  1) AI参与了哪个环节？
  2) 改善了哪个指标？
  3) 哪个责任仍由人承担？

### 3.5 总结页
- 固定三句：
  1) 本段建立了什么标准
  2) 标准如何用于项目
  3) 下一段将解决什么问题
- 每段结尾必须有宣言式一句话

## 4. Narration 风格规范（冻结）
- 口吻：导演式教学，不读稿，不空喊口号
- 句法：短句优先，结论先行，随后给证据
- 证据顺序：数据 > 图表 > 案例 > 观点
- 单页讲述格式：核心问题 + 互动动作 + 图表建议 + 页面节奏
- 禁用表述："AI颠覆一切"、"未来无限可能"等空泛句

## 5. Prompt 生成规范（冻结）
- Prompt 固定结构：
  `[Scene] + [Problem] + [Design strategy cues] + [Lighting/composition] + [Style] + [16:9]`
- 必含风格词：future academic cinematic, dark keynote, minimal
- Negative Prompt 必含：
  low resolution, blurry text, watermark, distorted hands, oversaturated colors, cluttered composition
- 同一章节共享 3–5 个固定风格词，确保视觉统一

## 6. 互动频率规范（冻结）
- 每8–10分钟至少1次互动
- 互动类型轮换：投票 / 找错 / 小组判断 / 口头复述 / 快速写卡
- 回答格式必须包含：对象 + 场景 + 判断依据

## 7. 40分钟课程节奏规范（冻结）
- 开场（10%）：4分钟
- 教材知识线（50–60%）：20–24分钟
- 设计文明线（20–25%）：8–10分钟
- AI未来线（10–15%）：4–6分钟
- 总结与作业（5–10%）：2–4分钟
