## 5) 页面动画建议（Animation）
- 慢节奏页：1步（标题淡入 0.4s）
- 中节奏页：2步（结论 0.25s → 图表 0.4s）
- 快节奏页：3步（冲突图 0.25s → 问题卡 0.25s → 反馈区 0.4s）
- 单页动画上限：3步；禁止复杂路径动画

## 6) 页面转场建议（Transition）
- 章节内：Fade 0.3s
- 章节切换（04→05、30→31、38→39、42→43）：Morph/Push 0.5s
- 禁用：旋转、翻页、弹跳等风格破坏型转场
