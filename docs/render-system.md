# 《设计概论》课程输出系统｜render-system

> 目标：把 `page-script.md / narration.md / prompts.md` 稳定转化为可交付 PPT。

## 1) page-script → PPT 页面映射规则

### 1.1 输入字段（来自 page-script）
- 页码
- 页面标题
- 核心问题
- 学生互动
- 图表建议
- 页面节奏（慢/中/快）

### 1.2 输出结构（PPT单页）
- Title Layer：页面标题 + 页码
- Core Layer：核心问题（1句）
- Evidence Layer：图表/案例/数据
- Interaction Layer：课堂互动指令
- Footer Layer：进度条 + 章节标记

### 1.3 转换模板
- 慢节奏页模板：标题大字 + 概念框 + 单图示
- 中节奏页模板：标题 + 证据图 + 解释要点
- 快节奏页模板：冲突图 + 问题卡 + 投票/判断区

---

## 2) narration → 备注讲稿映射规则

### 2.1 输入字段（来自 narration）
- 时间区段
- 讲述顺序（问题→证据→互动→结论）
- 互动触发语
- 结尾句/转场句

### 2.2 输出到PPT备注区
- 开场句（10–15秒）
- 主叙述（45–90秒）
- 互动指令（10–20秒）
- 收束句（10秒）

### 2.3 备注风格约束
- 短句，不写长段散文。
- 禁止空泛AI口号，必须对应具体环节与指标。

---

## 3) prompts → AI图像任务映射规则

### 3.1 Prompt拆分
- 场景对象（Scene）
- 问题目标（Problem）
- 策略线索（Design cues）
- 光影构图（Lighting/Composition）
- 统一风格（Style）
- 输出规格（16:9）

### 3.2 任务生成格式（可机读）
- Task ID：L03-P21-BG
- Slide Ref：Lesson03 Page21
- Asset Type：Background / Cutout / Texture
- Prompt：...
- Negative Prompt：...
- Priority：High / Medium / Low

### 3.3 验收标准
- 不出现水印、畸形手、模糊文字。
- 视觉风格与课程深色体系一致。

---

## 4) 图表 → SVG/信息图映射规则

### 4.1 图表类型白名单
- 三环图、双轴图、泳道图、矩阵图、对照表、流程图、热力图

### 4.2 生成规则
- 先定义数据字段，再选图表类型，不允许先画后补数据。
- 统一12列网格对齐。
- SVG优先（矢量可编辑），PNG仅用于复杂纹理底图。

### 4.3 命名规范
- `L03_P15_journey-map.svg`
- `L04_P28_buildability-score.svg`

---

## 5) 页面节奏 → 视觉节奏映射

- 慢节奏页：大标题 + 高留白 + 低动画（单步出现）
- 中节奏页：双层信息（主结论+证据图）+ 2步动画
- 快节奏页：冲突视觉 + 互动区域 + 3步以内动画
- 每2页必须发生一次视觉重构（版式/图表/全图切换）

---

## 6) 全局统一控制（冻结）

### 6.1 字体
- 中文标题：思源黑体 Heavy
- 中文正文：思源黑体 Regular
- 英文数字：Inter / SF Pro Display

### 6.2 色彩
- 背景：#0B1020 / #111A2E
- 文本：#F5F7FF / #A8B3CF
- 强调：#6EE7FF / #7C3AED / #22C55E / #F59E0B

### 6.3 留白
- 16:9（1920×1080）
- 安全边距：左右120，上下72
- 正文不超过6行

### 6.4 图层
- L1 标题层
- L2 证据层
- L3 互动层
- L4 导航层（页码/进度条）

### 6.5 动效
- 原则：服务叙事，不服务炫技
- 统一时长：0.25s / 0.4s / 0.6s
- 单页动画步数上限：3步

---

## 7) 产出目录规范（建议）
- `/renders/lesson03/slides/`
- `/renders/lesson03/assets/bg/`
- `/renders/lesson03/assets/svg/`
- `/renders/lesson03/notes/`
- lesson04 及后续同结构扩展

## 8) 质量门禁（Render QA）
- 结构门禁：标题/核心问题/证据/互动/页码是否齐全
- 视觉门禁：字体、色彩、留白是否符合冻结规范
- 叙事门禁：三线是否同题推进，是否存在空泛AI语言
- 技术门禁：文件命名、分辨率、导出格式是否合规
