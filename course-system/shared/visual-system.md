# 《设计概论》数字课程视觉系统 v1.0

## 1. 风格定位
- 关键词：未来学术感 / 电影感 / TED × Apple 发布会 × OpenAI DevDay。
- 情绪基调：理性、克制、前瞻、启发。
- 镜头语言：从宏观文明叙事推进到专业场景落点，再转向 AI 未来提案。

## 2. 字体系统
- 中文标题：思源黑体 Heavy（或 HarmonyOS Sans SC Bold）
- 中文正文：思源黑体 Regular
- 英文与数字：Inter / SF Pro Display
- 字号层级：
  - H1：72–96
  - H2：48–60
  - H3：32–40
  - Body：24–28
  - Caption：18–20

## 3. 深色背景体系
- 主背景：#0B1020（深蓝黑）
- 次背景：#111A2E（分层块）
- 强调渐变：#6EE7FF → #7C3AED
- 文字主色：#F5F7FF
- 次要文字：#A8B3CF
- 辅助线：#27324A

## 4. 页面节奏
- 40 分钟建议 24–28 页。
- 节奏结构：
  1) 开场提问（2页）
  2) 教材知识线（8页）
  3) 设计文明线（8页）
  4) AI未来线（8页）
  5) 总结与作业（2页）

## 5. 大标题结构
- 标题句式：
  - 结论先行："设计不是装饰，而是系统性决策"
  - 对比推进："从手工时代到算法时代"
  - 提问驱动："当 AI 会生成方案，设计师还做什么？"

## 6. 信息图层逻辑
- Layer 1：核心结论（1 句）
- Layer 2：证据（图表/案例/数据）
- Layer 3：专业映射（环境艺术设计情境）
- Layer 4：课堂互动（提问或小任务）

## 7. 页面边距
- 16:9 画布（1920×1080）
- 安全边距：左右 120 / 上下 72
- 网格：12 栏，栏距 24
