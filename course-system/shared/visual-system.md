# 《设计概论》课程视觉系统 v2.0

参考图：工作区 `40.jpg`。
适用范围：7次课、225页PPT，以及后续讲义、课程封面和案例卡片。

## 1. 视觉定位

### 1.1 核心气质

- 清晰：优先让学生看懂知识结构。
- 专业：接近现代设计咨询、学术展览和编辑设计。
- 开放：使用大面积白底，避免视觉压迫。
- 当代：用青蓝色块、图片裁切和模块化版面建立数字感。
- 克制：不使用过度科技化的霓虹、发光边框和复杂HUD界面。

### 1.2 从参考图继承的设计语言

1. 以白色和浅灰为主背景。
2. 以高饱和青蓝作为主要识别色。
3. 图片与纯色信息块形成明确的左右或上下切割。
4. 标题短、字号大，通常放在页面左上区域。
5. 用小型青蓝标签承载章节、概念和页码。
6. 每组内容中穿插全青蓝页和深灰反转页，形成节奏变化。
7. 案例页使用2—4张统一比例图片，而不是零散贴图。
8. 圆形肖像、六边形图标和编号圆点只作为少量功能组件。

### 1.3 针对《设计概论》的改造

- 医疗人物照改为设计师、作品、场所、器物和历史资料。
- 医疗服务图标改为设计分类、设计功能、流程和能力图标。
- 医疗团队卡片改为设计师、流派、案例或职业岗位卡片。
- 手术室深色页改为设计伦理、风险、争议和未来议题页。
- 医院介绍页改为教材概念、章节导读和方法说明页。

## 2. 色彩系统

### 2.1 主色板

| 角色 | 色值 | 用途 |
|---|---|---|
| 品牌青蓝 | `#14A3CA` | 大色块、章节标签、重点数字、图标 |
| 深青蓝 | `#0B718B` | 标题、深色强调、青蓝底上的层级区分 |
| 明亮青蓝 | `#35B9D5` | 渐变末端、图表高亮，不单独大面积使用 |
| 浅青蓝 | `#DDF2F6` | 提示框、表格浅底、图标背景 |
| 雾白 | `#F3F7F7` | 页面主背景 |
| 纯白 | `#FFFFFF` | 卡片、正文区、反转页文字 |
| 炭黑 | `#191D1F` | 深色反转页、主标题 |
| 正文灰 | `#526166` | 正文、图表标签 |
| 辅助灰 | `#91A0A4` | 次级说明、来源、未激活项 |
| 分割线 | `#D9E4E6` | 表格线、模块边界、页脚线 |

### 2.2 配色约束

- 不使用橙色、绿色等跳色作为课程辅助色。
- 所有强调、区分、警示和导航，统一从青蓝与灰阶内解决。
- 课程整体必须保持“青蓝主导”，避免每章各自发展成独立配色。

### 2.3 推荐色彩比例

- 白色或雾白：65%—75%。
- 青蓝：15%—25%。
- 图片：10%—35%，与白底比例联动。
- 炭黑：常规页不超过10%；深色反转页可达到70%—100%。
- 跳色：0%。

### 2.4 推荐渐变

- 青蓝渐变：`#0B718B → #14A3CA → #35B9D5`。
- 渐变方向：左下至右上，或左至右。
- 只用于封面、章节页和全色信息页。
- 正文页优先使用纯色，不使用多重渐变。

## 3. 字体系统

### 3.1 字体选择

本机已确认可用：

- 中文标题：HarmonyOS Sans SC Bold。
- 中文小标题：HarmonyOS Sans SC Medium。
- 中文正文：HarmonyOS Sans SC Regular。
- 英文与数字：Arial Bold / Arial Regular。

不使用未确认安装的字体，避免换机后版式漂移。

### 3.2 16:9画布字号

| 层级 | 字号 | 字重 | 使用场景 |
|---|---:|---|---|
| Cover | 54—64 pt | Bold | 课程总封面 |
| Section | 42—50 pt | Bold | 章节页、单元页 |
| H1 | 30—36 pt | Bold | 常规页标题 |
| H2 | 21—25 pt | Medium | 模块标题、卡片标题 |
| Body | 17—20 pt | Regular | 课堂正文 |
| Label | 11—13 pt | Medium | 青蓝标签、章节编号 |
| Caption | 9—11 pt | Regular | 来源、图片说明、页脚 |
| Number | 40—68 pt | Bold | 时间、步骤、核心数字 |

### 3.3 排版规则

- 中文标题字间距：0—1 pt，不使用大幅拉开字距。
- 英文标签可以使用全大写，字间距1.5—2.5 pt。
- 正文行距：1.25—1.4。
- 单个正文区最多6行；超过时拆页。
- 一行中文正文建议不超过28字。
- 标题尽量控制在16字以内，最多两行。
- 标题末尾通常不使用句号。

## 4. 画布与栅格

### 4.1 基础画布

- 比例：16:9。
- 设计基准：1920 × 1080 px。
- PowerPoint：13.333 × 7.5 in。

### 4.2 安全边距

- 左右：110 px。
- 上：78 px。
- 下：68 px。
- 图片允许出血到页面边缘，但文字必须留在安全区。

### 4.3 栅格

- 12栏栅格。
- 栏间距：24 px。
- 基础间距单位：12 px。
- 常用间距：12 / 24 / 36 / 48 / 72 / 96 px。
- 卡片内边距：28—36 px。
- 模块之间最小距离：24 px。

### 4.4 常用版面比例

- 文字40% + 图片60%。
- 图片45% + 文字55%。
- 左右50% + 50%。
- 主内容67% + 辅助内容33%。
- 三列卡片：每列约30%，间隔约5%。

## 5. 页面骨架

每一页固定包含以下区域中的3—5项：

1. 章节标签：左上，青蓝底白字或青蓝字。
2. 页面标题：标签下方或页面上方。
3. 核心内容：图片、图表、概念或案例。
4. 结论条：一句话总结，位于青蓝色块或页底。
5. 来源与页码：右下角或底部细线区域。

禁止每页同时出现超过两个视觉中心。

## 6. 教材可视化语法

核心原则：先判断教材知识关系，再选择图示形式；所有图解必须准确表达教材真实逻辑，不能制作只有视觉效果、没有知识含义的伪信息图。

### 6.1 内容与图示的匹配

| 内容类型 | 推荐表达形式 | 版面提示 |
|---|---|---|
| 定义、构成、分类 | 概念结构图、树状图、谱系图 | 优先T04、T05、T08 |
| 因果关系 | 因果链、关系网络、反馈回路图 | 优先T05、T09、T12 |
| 发展历程 | 时间轴、阶段演化图 | 优先T03、T07、T11 |
| 方法步骤 | 流程图、决策路径图 | 优先T05、T09、T10 |
| 空间层级 | 尺度嵌套图、地图、剖面图、空间模型 | 优先T04、T07、T10 |
| 对比内容 | 比较表、矩阵图、坐标图 | 优先T07、T08、T10 |
| 系统关系 | 系统模型、输入—过程—输出图 | 优先T05、T09 |
| 数据与结论 | 柱状图、折线图、比例图、重点数字 | 优先T09、T10 |
| 综合理论 | 知识地图、框架图 | 优先T07、T09、T11 |

### 6.2 图示绘制规则

- 一个图只表达一个核心判断，不把多种关系硬塞进同一页面。
- 图中箭头、层级、分组、时间顺序必须与教材叙述一致。
- 图示中的颜色只用于区分层级和模块，不替代逻辑本身。
- 如果教材没有数据，就不用假图表；改用结构图、流程图或对照表。
- 如果教材只有概念判断，没有严格因果证据，就不用强行画因果链。
- 图表中的关键词尽量短句化，避免再把整段教材搬进图形。

## 7. 十二种标准页面模板

### T01 课程总封面

- 构图：全幅设计作品或人物图，叠加青蓝半透明渐变。
- 标题：左侧或左下，白色，54—64 pt。
- 副标题：标题下方，16—20 pt。
- 适用：课程总封面，仅使用1次。
- 参考图对应：左上角全幅人物封面。

### T02 单次课封面

- 构图：左侧40%青蓝色块，右侧60%单张大图。
- 内容：课次、章节名称、核心问题。
- 青蓝块可使用低透明度线稿或教材页局部作为纹理。
- 适用：每次课第1页。

### T03 章节过渡页

- 构图：整页青蓝背景，左上大标题，右下小型章节编号。
- 可加入一条细白线或一个浅色几何图形。
- 正文不超过两行。
- 适用：四个40分钟单元的起始页。

### T04 左图右文概念页

- 左侧45%：一张纵向或方形图片。
- 右侧55%：标签、标题、定义和3个要点。
- 图片与文字区直接相接，不加厚重边框。
- 适用：教材定义、设计师介绍、历史事件。

### T05 左文右图概念页

- 左侧55%：标题、正文、结论。
- 右侧45%：一张大图或两张上下叠图。
- 可使用青蓝竖条连接标题与结论。
- 适用：方法说明、当下知识扩展。

### T06 半图半色案例页

- 左侧50%：案例大图。
- 右侧50%：青蓝底白字。
- 内容结构：案例名称、问题、策略、结果。
- 适用：经典案例和章节核心案例。
- 参考图对应：人体模型与青蓝说明页。

### T07 顶部大图 + 底部双栏

- 上方55%：横向大图。
- 下方45%：两个并列文字模块。
- 模块标题使用青蓝标签。
- 适用：作品观察、前后对比、材料与工艺。

### T08 三卡或四卡分类页

- 白底，顶部居中或左对齐标题。
- 三张或四张统一尺寸图片/图标卡片横向排列。
- 卡片下方依次为名称、定义、关键词。
- 适用：设计分类、功能类型、设计师、岗位、方法。

### T09 青蓝全底信息图页

- 整页品牌青蓝。
- 白色标题。
- 3—5个浅色圆形或六边形图标等距排列。
- 每个图标下方最多3行说明。
- 适用：课程框架、流程概览、功能归纳。

### T10 数据表格与对照页

- 白底，标题左对齐。
- 一侧为表格或矩阵，另一侧为场景图。
- 表头使用青蓝底白字；表格隔行使用浅青蓝。
- 适用：设计评价、方法比较、岗位能力、风险清单。

### T11 图集与作品集页

- 白底，大留白。
- 2—5张图片使用统一高度，可轻微错位但不旋转超过3度。
- 下方设置青蓝标题标签和一句总结。
- 适用：设计史作品、案例合集、学生练习展示。

### T12 深色反转议题页

- 炭黑背景，图片使用黑白或低饱和处理。
- 标题白色，关键词使用品牌青蓝。
- 可使用一张大图 + 2—3张小图的组合。
- 适用：设计伦理、风险、争议、失败案例、AI边界。
- 每8—12页最多出现1次，避免全课程重新变成深色系统。

## 8. 课程页面节奏

每个40分钟单元建议8页：

1. 单元过渡或问题页：T03 / T09。
2. 教材概念页：T04 / T05。
3. 教材概念页：T04 / T10。
4. 教材案例页：T06 / T07。
5. 当下连接页：T05 / T10。
6. 对比或分类页：T08 / T11。
7. 互动页：T09 / T12。
8. 单元总结页：T03的简化版本或T10。

节奏规则：

- 连续两页不能使用完全相同的图文比例。
- 连续三页不能都是白底纯文字。
- 每4—6页至少出现一次大图页。
- 每8—12页可出现一次深色反转页。
- 每个单元只允许一张全青蓝信息图页，避免青蓝过量。

## 9. 图片系统

### 8.1 图片内容优先级

1. 教材中的原始案例和作品。
2. 设计师、实物、建筑、空间和使用场景的真实照片。
3. 草图、图纸、模型、流程和材料细节。
4. 必要时生成的概念场景图。
5. 抽象装饰图形仅作背景，不作为主要证据。

### 8.2 图片色彩处理

- 白底页：保持自然色，允许轻微降低饱和度5%—15%。
- 青蓝页：图片可叠加品牌青蓝10%—25%透明蒙版。
- 深色页：图片改为黑白或低饱和，保留青蓝重点。
- 同一页图片应统一色温，不混用强暖光和强冷光。

### 8.3 图片裁切

- 人物：优先半身或工作状态，不使用证件照式居中大头照。
- 产品：保留完整轮廓，并补一张材料或使用细节。
- 建筑：优先展示人与尺度，避免只有无人航拍。
- 传达作品：保证文字与版面可辨认，不做过度透视。
- 历史资料：允许保留纸张质感，但背景需清理干净。

### 8.4 图片组合

- 两图：同宽并列，或主图2/3 + 辅图1/3。
- 三图：统一高度横排。
- 四图：2×2网格或四张竖图横排。
- 圆形裁切只用于人物、设计师和用户画像。
- 每页最多一种裁切形状。

## 10. 图标与几何组件

### 9.1 图标

- 使用2 px等视觉粗细的线性图标。
- 默认白色或深青蓝。
- 图标外框可使用圆形或六边形，不混用多种外框。
- 不使用拟物、3D塑料、彩色渐变图标。

### 9.2 标签

- 形状：小型横向矩形。
- 填充：品牌青蓝。
- 文字：白色，11—13 pt，Medium。
- 内边距：左右14—18 px，上下7—9 px。
- 文字长度：2—8个汉字。

### 9.3 编号圆点

- 直径：38—52 px。
- 青蓝底白字，或白底青蓝描边。
- 用于步骤、时间轴和课堂任务。
- 不用于普通项目符号。

### 9.4 装饰线

- 线宽：1—2 px。
- 颜色：分割线灰或品牌青蓝。
- 长度服务于对齐，不做无意义满页线框。

## 11. 图表与信息图

### 10.1 基本规则

- 数据图表优先使用白底。
- 主数据使用品牌青蓝，次数据使用浅青蓝和辅助灰。
- 只在风险或反例中使用暖橙。
- 删除不必要网格线、边框和图例背景。
- 图表标题直接陈述结论，不写“数据分析”。

### 10.2 推荐图表

- 时间轴：设计历程、职业发展。
- 矩阵：方法比较、岗位能力、风险判断。
- 流程图：设计程序、项目管理、PSS。
- 同心圆：需求、功能、利益相关者。
- 分层剖面：产品系统、环境尺度、界面结构。
- 前后对照：更新、优化、设计迭代。
- 卡片分类：设计领域、职业类型、创新类型。

### 10.3 禁止项

- 3D饼图。
- 彩虹色柱状图。
- 复杂阴影和发光。
- 一页同时出现三种不同图表。
- 为装饰而使用无数据意义的图表。

## 12. 表格系统

- 表头：品牌青蓝底，白字。
- 首列：深青蓝或Medium字重。
- 正文行：纯白与浅青蓝隔行。
- 边框：只保留必要横线，颜色`#D9E4E6`。
- 单元格内边距：12—16 px。
- 一页表格不超过6列、8行；超出时拆页。
- 表格正文不小于14 pt。

## 13. 课堂互动页

互动页采用以下两种样式：

### 12.1 青蓝问题页

- 全青蓝底。
- 左上标题：“请判断”“请比较”“请提出”。
- 中部放一个明确问题。
- 右下显示时间，如“3 MIN”。

### 12.2 深色争议页

- 深色背景 + 黑白案例图。
- 青蓝色标出冲突双方。
- 底部给出回答格式：“对象 + 场景 + 判断依据”。

互动页不同时展示答案，答案应在下一页揭示。

## 14. 页眉、页脚与编号

- 左下：课程名或章节简称，9—10 pt，辅助灰。
- 右下：两位数字页码，11—12 pt，深青蓝。
- 来源：位于图片下方或页脚上方，9—10 pt。
- 章节标签出现时，不再重复复杂页眉。
- 封面、章节过渡页可隐藏页脚。

## 15. 动画与转场

- 默认转场：淡化，0.3—0.5秒。
- 章节页可使用平滑或推进，但整套不超过两种转场。
- 列表使用按段落淡入，不逐字飞入。
- 流程图按步骤出现。
- 禁止弹跳、旋转、随机飞入和声音效果。
- 动画只服务讲述顺序，不用于装饰。

## 16. 七次课的识别方式

七次课不改变主色，只通过章节编号、图片主题和一个辅助色标记区分：

| 课次 | 主题 | 辅助标记 |
|---|---|---|
| 01 | 认识设计与设计历程 | 深青蓝细线 |
| 02 | 产品设计 | 青蓝实心圆 |
| 03 | 环境设计 | 浅青蓝底线 |
| 04 | 传达设计 | 青蓝方框 |
| 05 | 程序与方法 | 深青蓝编号 |
| 06 | 设计师与职业 | 圆形人物框 |
| 07 | 管理与创业 | 炭黑 + 青蓝标签 |

这些标记只用于导航，不形成七套独立配色。

## 17. 可访问性与可读性

- 白底正文使用炭黑或正文灰，不使用浅青蓝正文。
- 青蓝底正文使用纯白。
- 正文与背景保持高对比度。
- 不只依赖颜色表达正确、错误或分类，还要配合文字或图标。
- 投影环境下正文不小于17 pt。
- 关键图示必须有一句文字结论。
- 避免连续使用大面积纯青蓝，以减少视觉疲劳。

## 18. 禁止清单

- 禁止沿用旧版“全课程深蓝黑 + 霓虹渐变”的主风格。
- 禁止医疗十字、听诊器等参考图中的行业专属元素。
- 禁止一页堆放超过6张无统一比例的图片。
- 禁止大段教材原文直接贴入页面。
- 禁止同页出现超过2种字体。
- 禁止使用粗重投影、玻璃拟态和高亮发光边框。
- 禁止用装饰性英文替代中文信息。
- 禁止所有页面机械套用同一左右分栏。

## 19. 页面验收标准

每页导出前检查：

1. 是否只有一个明确结论？
2. 学生在3秒内能否看见标题和视觉中心？
3. 是否符合白底、青蓝和图片的主比例？
4. 图片是否提供证据，而非仅作装饰？
5. 正文是否不超过6行或已拆页？
6. 字号在教室投影中是否可读？
7. 是否与前后页形成版式变化？
8. 来源、页码和章节标记是否一致？
